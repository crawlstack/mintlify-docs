## Welcome to Crawlstack

Crawlstack allows you to run powerful web scrapers directly within your Chrome browser. It combines the ease of use of a browser extension with the power of a headless scraper, exposing a Public API and Webhook integrations.

## Key Features

- **Browser-based**: Runs in your own browser instance, using your cookies and session.
- **SQL Storage**: Data is stored in a local SQLite database (OPFS) or an external LibSQL database.
- **Deduplication**: Built-in support for deduplicating items and tasks.
- **Public API**: Expose your local browser as a remote scraping API.
- **Webhooks**: Push extracted data to any URL in real-time.
